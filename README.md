# Neighbourhood Event Management System

A full-stack web application for managing community events.

## Features
- User Registration and Login
- JWT Authentication
- Role-Based Access Control
- Event Creation and Management
- Event Registration
- Admin Dashboard
- Notifications

## Tech Stack
Frontend:
- ReactJS
- React Router
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation

### Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
npm start
